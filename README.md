# 프론트엔드 최윤지\_계산기 구현

- 사용 기술 및 버전
  - React 18
  - Typescript 5.5.3
  - TailwindCSS 3.4.6
  - Vite 5.3.4

---

## 🔥 구현 항목

- 사칙연산 기능 구현
  - 모든 사칙연산은 연속적으로 계산 할 수 있습니다.
  - 0 다음에 숫자를 입력할 시, 자동으로 숫자만 보이도록 구현. ex) 012 -> 12로 대체
- 삭제 버튼을 누르면 입력중인 숫자값이 하나씩 지워지도록 구현.
- 키보드로 사칙연산 기능 사용 가능하도록 구현.
- 값을 0으로 나누는 경우, 정의되지 않은 연산자를 입력하는 경우, 결과값이 무한대나 NaN으로 나오는 경우는 에러 처리를 하고 메세지를 띄웁니다.
